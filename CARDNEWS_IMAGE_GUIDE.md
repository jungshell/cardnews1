# 카드뉴스 이미지 생성 최적 가이드

## 🎯 추천 방법 (무료 + 초보자 친화)

### 방법 1: Canva 사용 (가장 추천 ⭐⭐⭐⭐⭐)

**장점:**
- ✅ 완전 무료 (기본 기능)
- ✅ 웹 브라우저만 있으면 사용 가능
- ✅ 템플릿이 많아 초보자도 쉽게 사용
- ✅ 드래그 앤 드롭으로 간단한 편집
- ✅ 브랜드 컬러 저장 가능
- ✅ 한글 폰트 지원

**사용 방법:**
1. [Canva.com](https://www.canva.com) 접속 (무료 회원가입)
2. "카드뉴스" 또는 "Instagram Post" 템플릿 검색
3. 템플릿 선택 후 편집:
   - **제목 입력**: 각 카드의 HEAD 내용
   - **본문 입력**: 각 카드의 BODY 내용
   - **배경색 변경**: 
     - Cover: #6750A4 (진한 파란색/보라색)
     - Program/Impact/Result: 밝은 회색/흰색
     - Closing: 연한 파란색/초록색
   - **아이콘 추가**: 
     - ZIP 파일에서 다운로드한 SVG 아이콘 업로드
     - 또는 Canva 내장 아이콘 검색 (IMAGE_KEY 키워드 사용)
   - **무료 이미지 추가**: 
     - Canva 내장 무료 이미지 검색
     - IMAGE_KEY 키워드로 검색 (예: "success", "drone", "AI")
4. 각 카드별로 반복하여 8장 제작
5. 다운로드: PNG 또는 JPG 형식 (1080x1080px 권장)

**팁:**
- 브랜드 컬러(#6750A4, #625B71)를 "색상"에 저장해두면 재사용 편리
- 템플릿을 복제하여 일관된 디자인 유지
- 한 카드를 완성한 후 복제하여 나머지 카드 제작

---

### 방법 2: Pexels + Canva 조합

**장점:**
- ✅ 완전 무료
- ✅ 고품질 무료 이미지 사용 가능
- ✅ 저작권 걱정 없음

**사용 방법:**
1. [Pexels.com](https://www.pexels.com) 접속
2. IMAGE_KEY 키워드로 검색 (예: "success", "drone", "AI")
3. 적합한 이미지 다운로드
4. Canva에서 이미지 업로드 후 카드뉴스 제작
5. 텍스트 오버레이 추가

---

### 방법 3: AI 이미지 생성 (고급)

**무료 옵션:**
- **Leonardo.ai** (무료 티어: 일일 150 크레딧)
- **Bing Image Creator** (Microsoft 계정 필요, 무료)
- **Stable Diffusion** (로컬 설치, 완전 무료)

**사용 방법:**
1. 시스템에서 제공하는 "AI 이미지 생성 프롬프트" 복사
2. AI 이미지 생성 사이트에 프롬프트 입력
3. 생성된 이미지 다운로드
4. Canva에서 이미지 + 텍스트 조합

**주의사항:**
- AI 생성 이미지의 상업적 사용 가능 여부 확인 필요
- 일부 서비스는 무료 티어에 제한이 있음

## 📋 현재 시스템에서 제공하는 자료

### 1. AI 이미지 생성 프롬프트
각 카드별로 제공되는 프롬프트를 복사하여 AI 이미지 생성기에 사용할 수 있습니다.

**포함 내용:**
- 브랜드 컬러 (#6750A4, #625B71)
- 카드 타입별 배경색 가이드
- 제목 및 본문 내용
- IMAGE_KEY 키워드

### 2. Iconify/Material Icons SVG
ZIP 파일로 다운로드 가능한 SVG 아이콘 파일들입니다.

**사용 방법:**
- Canva에서 SVG 파일 업로드
- 크기 및 색상 조정
- 카드뉴스에 배치

### 3. IMAGE_KEY 키워드
각 카드의 IMAGE_KEY를 활용하여:
- Canva 내장 이미지 검색
- Pexels 무료 이미지 검색
- AI 이미지 생성 프롬프트 작성

## 🎨 디자인 가이드라인

### 브랜드 컬러
- **Primary**: #6750A4 (진한 파란색/보라색)
- **Secondary**: #625B71 (회색/보라색)

### 카드 타입별 배경색
- **Cover**: 진한 파란색/보라색 계열 (#6750A4)
- **Program/Impact/Result**: 밝은 회색/흰색 계열
- **Closing**: 연한 파란색/초록색 계열

### 텍스트 스타일
- **제목(HEAD)**: 굵게, 큰 폰트 (24-32px)
- **본문(BODY)**: 일반, 중간 폰트 (16-20px)
- **폰트**: 깔끔한 한글 폰트 (예: 나눔고딕, Noto Sans KR)

### 레이아웃
- **비율**: 정사각형 (1:1, 1080x1080px 권장)
- **여백**: 충분한 여백으로 가독성 확보
- **정렬**: 중앙 정렬 또는 왼쪽 정렬

## 💡 실전 워크플로우 (Canva 기준)

### 1단계: 준비
1. ZIP 파일 다운로드 (아이콘 SVG 파일)
2. 각 카드의 HEAD, BODY, IMAGE_KEY 확인
3. AI 이미지 생성 프롬프트 복사 (필요시)

### 2단계: 템플릿 선택
1. Canva에서 "카드뉴스" 또는 "Instagram Post" 검색
2. 깔끔한 템플릿 선택 (1080x1080px)

### 3단계: 첫 번째 카드 제작
1. **배경색 설정**: 카드 타입에 맞는 색상 적용
2. **제목 입력**: HEAD 내용
3. **본문 입력**: BODY 내용 (있는 경우)
4. **아이콘 추가**: ZIP에서 다운로드한 SVG 또는 Canva 내장 아이콘
5. **이미지 추가** (선택): IMAGE_KEY로 검색한 무료 이미지

### 4단계: 나머지 카드 제작
1. 첫 번째 카드를 복제
2. 배경색, 제목, 본문만 변경
3. 아이콘/이미지는 카드 내용에 맞게 조정

### 5단계: 다운로드
1. 각 카드를 개별적으로 다운로드 (PNG 또는 JPG)
2. 또는 Canva에서 여러 페이지로 저장

## ⚠️ 주의사항

### 저작권
- ✅ Pexels, Unsplash: 상업적 사용 가능 (무료)

- ⚠️ AI 생성 이미지: 서비스별 라이선스 확인 필요
- ❌ 구글 이미지 검색 결과: 저작권 확인 필수

### 이미지 품질
- **해상도**: 최소 1080x1080px (SNS 업로드 권장)
- **파일 형식**: PNG (투명 배경) 또는 JPG
- **파일 크기**: 5MB 이하 권장 (SNS 업로드 제한)
